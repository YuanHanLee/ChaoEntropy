ChaoEntropy

  • Author: Y. T. Wang ; Y. H. Lee ; K. S. Tseng ; Anne Chao
  • Contributor: T. C. Hsieh
  • URL: http://chao.stat.nthu.edu.tw/blog/ The purpose of this R package ChaoEntropy is mainly to provide a new Shannon entropy estimator proposed by Chao et al. (2013) for both individual-based (abundance) data and sample-based (incidence) data.

Run the development version

  • Required: R
  • Suggested: Rstudio

Start R(studio) and copy-and-paste the commands below:

#install.packages('devtools')
library(devtools)
install_github('ChaoEntropy','JohnsonHsieh')
library(ChaoEntropy)
